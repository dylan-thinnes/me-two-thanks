About the Site
I wanted to make a site that prioritizes having a very small footprint, yet still looks nice and behaves like a single page app, and can gracefully drop back if JavaScript is disabled. I think I succeeded in that respect.

- When Javascript is disabled, links to different endpoints call upon Rails to render all markup server-side. Disable JavaScript to see for yourself how the site copes just fine.
- Aside from content and profile picture, the server sends only 4 kilobytes over the network after assets are minified and compressed.
- When Javascript is enabled, it takes over regular link behaviour and uses XHR and the History API instead, loading only the content you need.
